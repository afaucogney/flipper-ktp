Flipper Ktp is maintained by volunteers. Your help is welcome and will benefit the entire Android community!

Here's how you can help:

  • Contribute to Help Wanted issues.
  • Answer StackOverflow questions.
  • Provide feedback on pull requests.
  • Contribute code by forking the repository on GitHub and sending a pull request. Please read Dev Environment for Flipper Ktp contributors. When submitting code, please make every effort to follow existing conventions and style in order to keep the code as readable as possible.
